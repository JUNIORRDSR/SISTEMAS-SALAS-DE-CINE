# 🎬 Sistema de Salas de Cine

## Sistema de Información para Control y Administración de Taquillas en Salas de Cine

## 📋 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un **Sistema Integral de Gestión para Salas de Cine** que permite la administración completa de operaciones cinematográficas, desde la gestión de usuarios y películas hasta la venta de boletos y generación de reportes.

El sistema está diseñado para automatizar y optimizar los procesos de un cine, proporcionando herramientas específicas para diferentes tipos de usuarios (Administradores y Cajeros) y ofreciendo una experiencia fluida para los clientes en la reserva y compra de entradas.

## 🎯 Objetivos

- **Automatizar** el proceso de venta y reserva de boletos de cine
- **Gestionar** de manera eficiente la información de películas, horarios y salas
- **Controlar** el acceso mediante sistema de autenticación con roles diferenciados
- **Optimizar** la ocupación de salas mediante sistema de reservas inteligente
- **Generar** reportes analíticos para la toma de decisiones empresariales
- **Implementar** tecnologías de IA para recomendaciones personalizadas

## 👥 Tipos de Usuario

### 🔧 Administrador

- Gestión completa de cajeros y clientes (CRUD)
- Administración de películas y horarios
- Configuración de salas y asientos
- Acceso a reportes de ventas y auditoría
- Control total del sistema

### 💼 Cajero

- Visualización de cartelera y horarios
- Gestión de reservas y ventas de boletos
- Atención al cliente para compras
- Acceso limitado según permisos

### 🎪 Cliente

- Navegación por cartelera de películas
- Selección de horarios y asientos
- Reserva y compra de entradas
- Cancelación de reservas (antes de la función)

## 🚀 Funcionalidades Principales

### 📚 Gestión de Películas

- CRUD completo de películas (título, género, duración, clasificación)
- Asignación de horarios por sala
- Validación de conflictos de programación
- Control de disponibilidad

### 🎫 Sistema de Reservas y Ventas

- Selección interactiva de asientos por bloques y filas
- Reservas con tiempo límite configurable
- Generación automática de tickets
- Control de disponibilidad en tiempo real
- Anulación de reservas

### 📊 Reportes y Análisis

- Reportes de ventas por película y fecha
- Identificación de clientes VIP
- Auditoría de acciones de usuarios
- Exportación de reportes en PDF
- Análisis de ocupación de salas

### 🤖 Inteligencia Artificial

- Sistema de recomendaciones basado en historial
- Chatbot asistente para compras
- Análisis predictivo de preferencias

### 🔍 Funcionalidades Avanzadas

- Interfaz de búsqueda global (estilo Google)
- Sistema de notificaciones
- Optimización de interfaz (UI/UX)
- Generación de tickets automáticos
